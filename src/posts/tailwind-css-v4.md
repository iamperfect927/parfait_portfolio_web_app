# Tailwind CSS v4 is Here!

The latest version of Tailwind CSS brings significant improvements to performance and developer experience.

## Faster Build Times

The new engine is rewritten in Rust, making it lightning fast.

## Simplified Configuration

No more `tailwind.config.js` for basic setups! Use CSS variables natively.

### Example

```css
@theme {
  --color-primary: #ff0000;
}
```

Upgrade today and experience the speed.
